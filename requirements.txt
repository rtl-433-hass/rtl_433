# Runtime dependencies for the rtl_433 integration.
#
# These are installed by Home Assistant when the integration is loaded.
# See custom_components/rtl_433/manifest.json for the authoritative list.
#
# This integration intentionally has NO third-party runtime dependency: it
# consumes the rtl_433 HTTP server's WebSocket stream, parses the JSON events
# with the standard library, and maps fields to Home Assistant entities. The
# WebSocket client uses aiohttp and YAML parsing uses PyYAML / voluptuous,
# all of which Home Assistant core already provides.
#
# Keep this file empty unless a genuinely new runtime package is required.
